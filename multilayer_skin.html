Recherche du sous-arbre de Steiner contenant les proteines cibles

Voir https://github.com/krashkov/SteinerNet/blob/master/vignettes/tutorial.pdf pour le tutoriel de SteinerNet. Nos ‚Äúterminals‚Äù dans le sens du probl√®me de l‚Äôarbre de Steiner sont les n≈ìuds correspondant aux prot√©ines de l‚Äôenvironnement m√©tabolique proche. Comme on utilise une heuristique (SP = Shortest Path based approximation), le sous-r√©seau obtenu sera diff√©rent (en nombre de vertices et edges) a chaque fois qu‚Äôon ex√©cute le code suivant, a moins d‚Äôutiliser une graine al√©atoire fix√©e.

# On s'assure que la liste de terminaux ne contient pas des vertices non-pr√©sents dans le r√©seau humain

  Entry %in% biogrid$A | Entry %in% biogrid$B
